Architecture

Four systems. One organism.

SPINA

Blockchain memory backbone. Every network event is hashed into an append-only, cryptographically verifiable chain. Replay the last hour, day, or month to trace exactly what changed.

from nova_spina.chain import SpinaChain

Cytokine

Digital immune system. Learns baseline behavior from passive observation and scores deviations. No rules, no thresholds, no signatures. Distinguishes legitimate topology changes from intrusion.

from nova_cytokine.engine import CytokineEngine

Synapse

Inter-agent connection mesh. Agents discover each other and share intelligence via lightweight pub/sub with cryptographic identity. One agent detects an anomaly — all agents are immunized.

from nova_synapse.engine import SynapseEngine

Cockpit

Real-time 3D organism visualization. Every device, link, and anomaly rendered as a living cell in a molecular network. Tilt, zoom, click to auscultate any node.

nova cockpit --port 8080

Quick Start

CLI

# Passive scan — zero packets sent
nova scan --subnet 192.168.1.0/24

# Immune engine — continuous watch
nova immune --watch

# 3D cockpit
nova cockpit --port 8080

Python API

from nova_spina.chain import SpinaChain
from nova_cytokine.engine import CytokineEngine
from nova_synapse.engine import SynapseEngine

synapse = SynapseEngine()
router = synapse.discover_organ(
    "192.168.1.1", "aa:bb:cc:dd:ee:ff")
webserver = synapse.discover_organ(
    "192.168.1.10", "11:22:33:44:55:66")

synapse.form_synapse(router, webserver, "TCP", 443)

spina = SpinaChain()
spina.add_block({"scan": "complete"})

cytokine = CytokineEngine(synapse)
alerts = cytokine.check()
for a in alerts:
    print(f"[!] {a.message}")
